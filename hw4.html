List Usage

Tips Usage
Unordered List An ordered list is created using the <ul> tag.
Ordered List An ordered list is created by using the <dl> tag. It is used to present a list of terms and their corresponding definitions. Each term is represented by the <dt> tag, and each definition is represented by the <dd> tag.
Definition List A definition list is created using the <dl> tag. It is used to present a list of terms and their corresponding definitions. Each term is represented by the <dt> tag, and each definition is represented by the <dd> tag.
